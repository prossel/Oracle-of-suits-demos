Active Area Calibration

This sketch demonstrates how to calibrate an on-screen active area using MediaPipe Hands and p5.js. Use P (position) or S (size) keys with arrow keys to adjust the cyan rectangle. The index fingertip shows both canvas coordinates (yellow) and mapped coordinates within the active area (cyan). Press C to toggle calibration mode. Settings are automatically saved. Allow camera access when prompted. Works best in a secure context (https or localhost).
